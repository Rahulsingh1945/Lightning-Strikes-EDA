# Lightning Strikes Exploratory Data Analysis (2016–2018)

## Overview
This project analyzes lightning strike data from NOAA to identify
geographic and temporal patterns across the years 2016–2018.

## Key Analysis
- Locations with highest strikes in a single day
- Locations with most strike-active days
- Lightning distribution by weekday
- Monthly percentage contribution of lightning strikes by year

## Visualization
The final visualization compares monthly lightning strike percentages
across years using grouped bar charts.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
